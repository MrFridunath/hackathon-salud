# hackathon-salud
Bot desarrollado en el Hackathon de Salud de Junio de 2019

## PRE REQUISITES
[Node 10+][1]

## INSTALL
```
npm install
````
## RUN
```
node server.js
```
## TRY WITH BOT EMULATOR
[BotFramework-Emulator 4+][2]

[2]: https://github.com/Microsoft/BotFramework-Emulator/releases/latest "BotFramework-Emulator"
